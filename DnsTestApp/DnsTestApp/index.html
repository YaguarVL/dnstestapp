Произошла ошибка подключения к базе данных. Проверьте корректность строки подключения в файле web.config

Для реализации задания был использован пустой макет веб приложения. MVC или WebForms не использовались, так как о них не упоминалось в тексте задания, а для такого небольшого приложения они являются избыточными.

Управление деревом осуществляется с помощью jQuery и плагина jsTree. Поскольку раньше мне не приходилось использовать деревья c загрузкой данных по Ajax, и у меня не было предпочтений относительно библиотек их реализующих, то была выбрана первая встреченная в поисковике библиотека с неплохими отзывами и удовлетворяющая следующим требованиям:

  1. Загрузка данных по Ajax
  2. Возможность "Drag and Drop"
  3. Возможность реализовать папки и файлы (файл не может быть контейнером)
  4. Отображение процесса загрузки спиннером
Добавление новых данных из пользовательского интерфейса не предусмотрено.

Субд MsSQL 2012 Express. ORM - Entity Framework. EF - достаточно тяжёлое расширение, но его использование в таком небольшом приложении оправданно тем, что не нужно вручную писать (и отлаживать) SQL запросы. К тому же, с его помощью очень удобно задавать начальные данные для базы данных.
